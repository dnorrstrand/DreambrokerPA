Dreambroker Programming Assignment


This is a web interface for testing the functionality of an API, which analyzes text srtings and returns statistics. You can enter any text string into the input box below and press "ANALYZE". The API will return your text stats in the same window.

The following CURL command can be used to test the API:

curl --location --request POST 'https://young-dusk-81991.herokuapp.com/analyze' \
--header 'Content-Type: application/json' \
--data-raw '{"text":"Hello, world! Its a rainy day!!"}'
